Extensions/Snippets/GetCodeSnippet.PowerCode.ps1
------------------------------------------------




### Synopsis
Gets the list of snippets files



---


### Description

Gets the list of snippets files in all packages Visual Studio Code.



---


### Examples
> EXAMPLE 1

```PowerShell
Get-PowerCode -CodeSnippet
```


---


### Parameters
#### **CodeSnippet**

If set, will get a list of all code snippet files.






|Type      |Required|Position|PipelineInput|Aliases     |
|----------|--------|--------|-------------|------------|
|`[Switch]`|true    |named   |false        |CodeSnippets|





---


### Syntax
```PowerShell
Extensions/Snippets/GetCodeSnippet.PowerCode.ps1 -CodeSnippet [<CommonParameters>]
```
